# silo

experimental document based access for boltdb

```json
{
  "users": {
    "id": 0,
    "name": "",
    "contact": {
      "primary": "",
      "secondary": ""
    }
  }
}
```

create /users/ {payload}
payload as below

```json
{
  "name": "name",
  "contact": {
    "primary": "",
    "secondary": ""
  }
}
```
